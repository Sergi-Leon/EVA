Hexadecimal y hexadecimal transparente

El sistema hexadecimal es el sistema de numeración posicional que tiene como base el 16. El sistema de colores hexadecimal está representado por una serie de 6 números que respectivamente representan a la intensidad de: Rojo (#FF0000), Azul (#00FF00) y Verde (#0000FF). El sistema hexadecimal transparente tiene la base la misma que el sistema hexadecimal, pero agregando dos cifras al total de 6 para interpretar la transparencia de la misma, por ejemplo: #FFFFFFFF.

RGB y RGBA

RGB se compone por los 3 colores primarios (Rojo, Verde y Azul). RGB se trata de un modelo mediante el cual es capaz de representar distintos colores a partir de la mezcla de los tres colores primarios.​ Está dividido en 3 columnas, y en cada columna hay un número del 0 hasta el máximo que es el 255. La primera columna indica el color Rojo, la segunda columna indica el color Verde, y la tercera columna indica el color Azul. RGB se basa en la cantidad de iluminación que se le aplica a estos tres colores primarios, y dependiendo de dicha iluminación conseguiremos un color u otro. RGBA tiene la misma función que el RGB, pero añadiendo el canal "Alpha". La función de dicho canal es indicar la opacidad de cada píxel, es decir, podemos ajustar a nuestro gusto el nivel de transparencia del color.​ Al añadirle el Alpha, crea otra columna para definir el nivel de transparencia.​ Para visualizar el color RGB y el color RGBA es a través de números decimales.
